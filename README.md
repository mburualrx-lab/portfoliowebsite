# Alex's Personal Website

This is my personal portfolio website where I showcase my projects.

## How to View

Open `index.html` in your web browser to view the website.

## Customization

- Edit `index.html` to change the content, such as adding more projects or updating contact info.
- Modify `style.css` to change the appearance.

## Adding Projects

In the `#projects` section, add more `<div class="project">` elements with your project details.

## Hosting

To host this website online, you can use services like GitHub Pages, Netlify, or Vercel. Simply upload these files to your chosen platform.

## Troubleshooting

- If styles don't load, ensure `style.css` is in the same directory as `ALEX.html`.
- For responsive design issues, check the CSS grid and media queries.
